224      the lines after it. 
225   
226      I'm sure there is a better way, but given the structure of a 
227      reStructuredText document, let's take a naive approach and scan the file 
228      to update the lines starting with ``.. restblog::`` and the contiguous lines 
229      before an empty line with the given metadata, e.g.:: 
230       
231          .. restblog:: 
232             :title: Some title here 
233             :source: yes 
234   
235      Parameters: 
236   
237      - file_name: File to the source post in reStructuredText to split. 
238   
239      Returns a tuple of three lists with the lines before the directive, the 
240      directive itself and after it. 
241      ''' 
242       
243      file = open( file_name, 'r' ) 
244      lines = file.readlines() 
245      file.close() 
246   
247      before = [] 
248      restblog = [] 
249      after = [] 
250   
251      extracting = False 
252   
253      for index, line in enumerate( lines ): 
254          if not extracting: 
255              if '.. restblog::' in line: 
256                  extracting = True 
257                  restblog.append( line ) 
258              else: 
259                  before.append( line ) 
260          else:

263              else: 
264                  # we've reached the end of the directive as indicated by an 
265                  # empty line. 
266                  extracting = False 
267                  after = lines[index+1:] 
268                  break
