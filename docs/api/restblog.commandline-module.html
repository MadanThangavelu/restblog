getCommandsRoot()

source code 
Returns absolute path to the commands directory inside the restblog package. Note that this is not a Python subpackage, it is just a plain directory.
Returns: str

getCommandNames()

source code 
Returns a list of string with then names of all the subcommands available for the restblog executable. Subcommands are simple Python modules inside the restblog/commands directory.
Returns: list

loadCommandByName(name)

source code 

Locates and imports a restblog subcommand by the given name.

Subcommands can be imported programmatically like this:

>>> import restblog.commandline
>>> command = restblog.commandline.loadCommandByName( 'list' )
>>> command.run( '--last=10' )

Returns a module object.

Returns: module
